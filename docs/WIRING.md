# 🐱 DeskCat-Nano 云台+轮子版本 接线图

## 硬件清单

| 组件 | 数量 | 说明 |
|------|:----:|------|
| ESP32-S3 Nano | 1 | 主控 (PSRAM 8MB, Flash 16MB) |
| SG90 舵机 | 2 | 云台: Yaw(水平) + Pitch(俯仰) |
| TB6612FNG 驱动板 | 1 | 驱动两个直流减速电机 |
| 直流减速电机 | 2 | 左右轮子 |
| 5V 电源 | 1 | 给舵机和电机供电 (2A+) |
| USB-C 线 | 1 | 给 ESP32 供电+上传程序 (数据线) |

---

## 接线总表

```
┌─────────────────────────────────────────────────────────┐
│                   ESP32-S3 Nano                          │
├─────────────────────────────────────────────────────────┤
│ USB-C ─── 电脑 (供电 + 上传程序)                          │
│ 3.3V ─── 所有外设 VCC (舵机除外!)                        │
│ GND  ─── 所有外设 GND + 外部电源 GND (共地!)              │
└─────────────────────────────────────────────────────────┘
```

### 1️⃣ 云台舵机 × 2 (SG90)

| 舵机 | 功能 | ESP32 引脚 | 信号线颜色 |
|------|------|:---------:|:---------:|
| SG90 #1 | 水平/Yaw (左右转) | **GPIO4** | 橙/白 |
| SG90 #2 | 俯仰/Pitch (上下转) | **GPIO5** | 橙/白 |

**接线：**
```
舵机(橙/白) → GPIO4(Yaw) / GPIO5(Pitch)
舵机(红)    → 外部 5V 电源正极
舵机(棕)    → 外部电源 GND ←→ ESP32 GND (共地!)
```

### 2️⃣ 轮子电机 × 2 (TB6612FNG)

| 电机 | 功能 | 驱动引脚 | ESP32 |
|:----:|------|:--------:|:-----:|
| 左轮 | 前进/后退 | IN1 | **GPIO6** |
| | | IN2 | **GPIO7** |
| | | PWM | **GPIO8** |
| 右轮 | 前进/后退 | IN3 | **GPIO15** |
| | | IN4 | **GPIO16** |
| | | PWM | **GPIO17** |

**TB6612FNG 接线：**
```
TB6612FNG        ←→  ESP32 / 电源
VM(电机电源)     ←→  外部 5V (不通过 ESP32!)
VCC(逻辑电源)    ←→  ESP32 3.3V
GND              ←→  外部电源 GND + ESP32 GND
AO1/AO2          ←→  左轮电机
BO1/BO2          ←→  右轮电机
STBY             ←→  ESP32 3.3V (始终使能)
```

### 3️⃣ 板载 LED

| 引脚 | 功能 |
|:----:|:----:|
| **GPIO48** | 心跳指示 (已连接=快闪, 未连接=慢闪) |

### 4️⃣ 电源分配

```
USB 5V ─── ESP32 (通过 USB-C)
                │
                ├── ESP32 3.3V ─── TB6612 VCC + 舵机逻辑(不推荐)
                │
外部 5V/2A+ ─── 舵机 VCC(红)
             ─── TB6612 VM
             ─── GND ─── ESP32 GND (共地)
```

## 共地示意图

```
      ┌─────────────┐
      │   5V 电源    │
      │  (2A+)      │
      ├─────────────┤
      │ VCC(+) ─────┼── 舵机红 ── TB6612 VM
      │ GND(-) ─────┼── 舵机棕 ── TB6612 GND ── ESP32 GND
      └─────────────┘
```

**重要：所有 GND 必须连在一起！** 否则舵机会乱转，电机不转。

---

## 快速检查清单

- [ ] 舵机信号线接对了 GPIO4/GPIO5？
- [ ] 舵机 VCC(红) 接的是**外部 5V**，不是 ESP32？
- [ ] 所有 GND 共地了？
- [ ] TB6612 STBY 接了 3.3V？
- [ ] 外部电源够 2A 以上？
- [ ] USB 线是**数据线**？

---

## 上传指南

1. 按住 **BOOT** 键 → 不松手
2. 点 PlatformIO → Upload
3. 看到 `Connecting...` 松手
4. 上传完成自动重启
5. 手机 App 扫描 BLE "DeskCat-Nano" 即可连接
